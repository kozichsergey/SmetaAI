СПИСОК ПОЗИЦИЙ:

ПРАВИЛА ГРУППИРОВАНИЯ:

1. **ОБОРУДОВАНИЕ И МАТЕРИАЛЫ:**
   - Группируй по типу: вентиляторы, трубы, кабели, крепеж и т.д.
   - Учитывай технические характеристики: диаметр, мощность, сечение, материал
   - Пример: "Вентилятор канальный 315мм" + "Вентилятор круглый Ф315" = один кластер

2. **РАБОТЫ И УСЛУГИ:**
   - Группируй по виду работ: монтаж, демонтаж, наладка, проектирование
   - Учитывай объект работ: что монтируется/демонтируется
   - Пример: "Монтаж вентилятора" + "Установка вентилятора" = один кластер

3. **СИСТЕМЫ И КОМПЛЕКСЫ:**
   - Группируй по назначению: вентиляция, отопление, водоснабжение, электрика
   - Учитывай подсистемы: приточная/вытяжная вентиляция, силовая/слаботочная электрика

4. **ИГНОРИРУЙ РАЗЛИЧИЯ В:**
   - Порядке слов ("Вентилятор канальный" = "Канальный вентилятор")
   - Сокращениях ("ф." = "диаметр", "шт" = "штука")
   - Незначительных вариациях названий брендов

5. **НЕ ГРУППИРУЙ ВМЕСТЕ:**
   - Разные типы оборудования (вентилятор ≠ насос)
   - Существенно разные характеристики (Ф100 ≠ Ф315)
   - Разные виды работ (монтаж ≠ демонтаж)

ФОРМАТ ОТВЕТА:
Верни JSON объект, где:
- Ключ = каноническое (лучшее) название для группы
- Значение = массив всех позиций из этой группы

Пример:
{
  "Вентилятор канальный круглый Ф315мм": [
    "1. Вентилятор круглый канальный Ф315мм.",
    "5. Вентилятор канальный диаметр 315"
  ],
  "Шумоглушитель круглый 315мм": [
    "2. Шумоглушитель 315 900",
    "8. Глушитель шума Ф315"
  ]
}

ВАЖНО: Отвечай ТОЛЬКО чистым JSON, без markdown обертки и комментариев!
